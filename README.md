# AWS Core Foundations

This repository is a **structured learning path for AWS core services**, designed for:
- Software Engineers (SWE)
- AI / ML Engineers
- Anyone preparing for **AWS-based technical interviews**

The repo combines **clear theory** with **hands-on practice guides**, covering real-world AWS workflows step by step.


## What to expect

By going through this repository, one can understand and practice:

- AWS Compute (EC2, ECS, Lambda)
- Networking (VPC, Subnets, Routing, Load Balancers)
- Storage (S3, ECR)
- Monitoring (CloudWatch)
- Data & ML Foundations (Glue, SageMaker)
- CLI-based AWS workflows
- End-to-end application deployment on AWS

This repository focuses on **core concepts** and **used in real production systems**.
